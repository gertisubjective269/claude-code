- Make sure the download finished

- Try opening the `.exe` again from the extracted folder
- Right-click the file and choose Run as administrator if needed

### Windows blocks the app

- Open the file details and confirm the download source
- Try again after the download finishes
- Move the file out of a protected folder
- Check if your antivirus placed it in quarantine

### The app opens but shows nothing

- Open the source folder again
- Check that the folder still contains files
- Use the refresh option if the app has one
- Pick another file from the list

### Search does not find results
